# 「写给你的信」— 网页解密游戏 GDD

## 基本信息

| 项目 | 内容 |
|------|------|
| 名称 | 写给你的信 |
| 主题 | 亲情 / 父子 / 职场霸凌 |
| 伪装形态 | 2010年左右功能机界面 |
| 时长 | 45-90分钟 |
| 难度 | 中等 |
| 平台 | 纯网页 (HTML/CSS/JS) |

## Fun Hypothesis

这游戏好玩的核心是：你在一个已故父亲的旧手机里翻找线索，最终发现他生前一直在用一种只有你才能看懂的方式，偷偷帮你收集了职场霸凌的证据，帮你打赢了劳动仲裁——但他没来得及告诉你。

## Design Pillars

1. **私密感** — 每个界面都在暗示「这是一个真实的人的私人物品」，玩家像在翻遗物
2. **延迟理解** — 前期看到的碎片在后期才会拼出完整意义，制造「原来他一直……」的情感冲击
3. **密码即记忆** — 所有加密/密码都基于父子共同记忆，不是随机数学谜题
4. **不煽情** — 情感靠事实传递，不靠BGM和文案。一条超市购物清单比一段独白更催泪

## 故事梗概

父亲老周（周建国），58岁，退休工人，独居。孩子周遥在外地工作，父子联系渐少。

周遥在公司遭遇职场霸凌（强制加班、克扣工资、言语侮辱），但从未告诉父亲。父亲偶然从孩子的一条朋友圈（短信提及）察觉异样，开始偷偷调查。

父亲瞒着孩子去公司附近蹲守、和前员工见面、记录加班情况、收集证据。他把所有证据整理成文档，写了一封信准备寄给孩子——但在寄出前因心梗去世。

孩子在整理遗物时打开了这部旧手机。

## 解密链路（三层）

### 第一层：翻找期（0-25min）

- **锁屏密码**：孩子的生日。线索：短信里父亲说「你生日那天给你发了红包」→ 日历中找到标注日期 → 密码是0915
- 自由翻阅短信、相册、备忘录
- 表面故事：父亲独居，和孩子的联系越来越少，最后一条短信是3个月前孩子回复的「嗯知道了」
- **隐藏线索**：备忘录有加密条目，密码提示「你小学二年级转学那天穿的什么颜色衣服」→ 相册里有转学合影，孩子穿红色 → 密码：红色/RED

### 第二层：拼合期（25-55min）

- 解开第一个加密备忘录，发现父亲在调查孩子的职场情况
- 短信记录里有和陌生号码的对话（前员工、工会的人），语气客气但暗含目的
- 通话记录显示某段时间频繁联系一个号码（劳动仲裁咨询热线）
- 日历标注了几个「见面」日期
- 音乐播放器里一份歌单，歌名首字母拼出信息 → 指向文件管理器

### 第三层：真相（55-90min）

- 解开最终加密文件，发现：
  - 一份完整的证据整理文档（加班记录截图、工资条对比、证人证言）
  - 一封没有寄出的信，内容极其务实：「我帮你查了这些，你可以这样维权……」
  - 最后一行附了超市小票照片，上面多买了一箱孩子爱喝的牛奶——「路过超市顺便买的，你回来的时候喝」

## 失败信号

| 信号 | 说明 | 检测方式 |
|------|------|----------|
| 锁屏卡住 | 密码太难 | 5分钟未进入主界面 |
| 翻找期无聊 | 内容量不够 | 短信停留 < 30秒 |
| 情感冲击不够 | 最终信件没有「原来如此」感 | playtest后问「你猜父亲在做什么」 |
| 密码太抽象 | 共同记忆不可解 | 每个密码至少2条线索 |

## 技术方案

- 前端：纯HTML/CSS/JS，模拟功能机界面
- 状态：localStorage存进度
- 控制台彩蛋：console.log('这不是你的手机。')
- 文件结构：index.html + css/style.css + js/app.js + js/data.js
